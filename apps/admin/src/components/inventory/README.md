# Inventory Components

Components for the Inventory Management module.

## Structure

This directory will contain:
- Supplier components (forms, lists, cards)
- Product components (forms, lists, cards)
- Inventory entry components (forms, item lists)
- Purchase order components (forms, order lists)
- Analytics dashboards

## Pattern

Follow the established component patterns in `components/hr/` and `components/accounting/`.
