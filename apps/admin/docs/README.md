## Conventions de Documentation

1. **Langue** : Documentation en anglais (code) ou francais (explications)
2. **Format** : Fichiers Markdown
3. **Structure** : Titres clairs, sections distinctes
4. **Exemples** : Code blocks avec syntaxe highlighting
5. **Mise a jour** : Date de derniere modification en haut du fichier
